##PhalApi框架扩展类库
此部分类库为 [PhalAPi框架](http://git.oschina.net/dogstar/PhalApi) 下的扩展类库包，各个扩展包各自独立，可以根据需要自动下载安装。

##目前已开源提供的扩展类库
###微信开发
此扩展可用于微信的服务号、订阅号、设备号等功能开发，则PhalApi框架下简单配置即可开发使用。
